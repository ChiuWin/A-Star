# A*算法巡飞弹轨迹

> 本项目由 ChiuWin 开发  

## 使用方法

### 1. 直接运行

- 直接运行 `demo.py`
- 在 `simulation_results` 文件夹中查看结果

### 2. 批量生成结果代码使用说明

- 修改 `run.py` 中的 `times` 为预期的循环次数
- 运行 `run.py` 文件
- 在 `simulation_results` 文件夹中查看结果
